LMath

   This program is used to design single layer solenoid air core inductors for things like antenna matching networks and filters. It works with most hf-vhf coils ranges. Since I don't have much access to torrid cores here in the middle of nowhere, air core will take care of most of what I need as long as you remember things like mutual-coupling with multiple coils. Faraday shields may be needed. Used along side programs like Elsie, it takes care of most of the math for you. Elsie™ is a lumped-element ("L-C") electrical filter design and network analysis program. Matching networks and filters become easy.

It's for Windows.

MS-Dos Keyboard to CW (Morse Code) Generator with RS-232 Keying

   I wanted to learn C programming and to learn morse code, so this is what I came up with to start with. This program is a keyboard to morse code generator. It's a command line program that uses control characters.

  • ; - send the keyboard inputs to the output and key the transmitter using rs-232 serial port.

  • & - settings menu.

  • " - exit the program.

It encodes characters
A-Z  0-9  !   #  '  ,  -  .   /  :  ?  @  $
into morsecode.
   It comes with a sample settings file in the .zip that needs to be in the same folder as the .exe is in. The program will run without it, but it will ask you 4 questions that are needed for the preset buffers and then it will create a new file. The program will create a log file that can be turned on/off in the settings. It has a sidetone that can be disabled in the settings also. You can also turn on/off the displaying of the morsecode to the display.

It's for Windows command line.

Windows Keyboard to CW (Morse Code) Generator with RS-232 Keying

   This program is a keyboard to morse code generator. It uses buttons and menu's instead of command characters. The keyboard can control everything except min/max the screen. Use the tab key to access the buttons, and alt key for menues. It encodes characters
A-Z  0-9   !  #  '  ,  -  .  /   :  ?  @  $
into morsecode. It uses rs-232 port to key the transmitter. It has 5 lables to display the status of the settings. It has 10 preset buffers. The six of the command line version plus 4 custom buffers that can be programmed in the settings menu.
   It comes with a sample settings file in the .zip that needs to be in the same folder as the .exe is in. The program will run without it, but it will ask you 4 questions that are needed for the preset buffers and then it will create a new file. The program will create a log file that can be turned on/off in the settings. It has a sidetone that can be disabled in the settings also. You can also turn on/off the displaying of the morsecode to the display.

It's for Windows.
