Detailed RS-232 D-Sub 9 Pins Description


   In most applications, only pins 2, 3, 5 on the D-Sub 9 Pins connector is used for the RS-232 application. This 3-wire communication mode is non-hardware-handshaking or asynchronous data communication. Meaning that the serial data is sent out based on a pre-set timing clock. (This is where USB came from.) The rest of the pins are meant for controlling the flow of the data. Hardware handshaking (synchronous data communication) is slower. These are hardly used these days because of the faster processing power of IC chips and the timing precision that we have now. For most applications, you don’t need to wire up these wires. But we will still be presenting the purpose of these pins on the D-sub connector for complete documentation of the pin out.

  • Pin 1 DCD – Data Carrier Detect (IN), indicator signal is detected from the remote terminal (the other end of the terminal).

  • Pin 2 RXD – Received Data (IN), data coming in from the remote terminal.

  • Pin 3 TXD – Transmitted Data (OUT), data sending out to the remote terminal.

  • Pin 4 DTR – Data Terminal Ready (OUT), local terminal side is ready to receive data.

  • Pin 5 GND – Signal Ground (COMMON), reference pin for all the signal pins.

  • Pin 6 DSR – Data Set Ready (IN), remote terminal is ready to receive and send data.

  • Pin 7 RTS – Request To Send (OUT), the local terminal is requesting the remote terminal to send over data.

  • Pin 8 CTS – Clear To Send (IN), remote terminal is ready to receive data.

  • Pin 9 RI – Ring Indicator (IN), the local terminal receives the ring from remote terminal.

MS-Dos 3 LED Controller using RS-232


   A simple program to toggle the LED's on/off. The majority of the program is for selecting the serial port to use. When the program first opens, it asks for the port number. If the port opens it continues on to the command characters input. If the program come back with the port doesn't exist it exits.
Commands

  • d: toggle the DTR LED(pin 4) on/off

  • r: toggle the RTS LED(pin 7) on/off

  • t: toggle the TXD LED(pin 3) on/off

  • x: exit the program.

Pin numbers are for the db-9 connector.

It's for Windows command line.
